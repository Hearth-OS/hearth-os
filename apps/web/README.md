# @hearth-os/web — Frontend (Person C)

Next.js app with Mapbox GL JS.

## Dev

```bash
npm run dev   # http://localhost:3000
```

## Key files

- `src/app/page.tsx` — main map page
- Add Mapbox component under `src/components/Map.tsx`
- Toggles + time slider under `src/components/Controls.tsx`
- Property drill-down under `src/components/PropertyPanel.tsx`
